* SQL 기초 3

      1. 정상 쿼리 : DISTINCT는 Alias 가능
      2. 쿼리 순서 : From - Where - Select - Order By

      SELECT DISTINCT job, sal+comm as "정렬쓰"
      FROM emp
      ORDER BY "정렬쓰";

      // 에러발생 : DISTINCT 키워드를 사용했을 때는 SELECT절에 있는 칼럼으로만
      SELECT DISTINCT job, sal+comm  
      FROM emp
      ORDER BY sal;
